# Release v1.0.0 - Cimientos del Sistema, Auth & CRM Básico

**Fecha de Lanzamiento:** 2026-06-09  
**Estado:** Completado y Desplegado Localmente

---

## Resumen del Lanzamiento
Esta versión establece las bases técnicas de Solar Hub, incluyendo el layout adaptable de la aplicación (Orion layout), el sistema de autenticación multi-tenant de Supabase y el módulo básico de gestión de clientes (CRM).

---

## Nuevas Características

### 1. Interfaz Adaptativa (Orion Layout)
- **Barra Lateral (Sidebar):** Menú de navegación fijo en desktop y colapsable en pantallas móviles.
- **Modo Campo:** Diseñado específicamente para técnicos que operan al aire libre con guantes y baja visibilidad solar.
  - Componentes táctiles grandes con altura mínima de `h-12` o padding `p-4`.
  - Colores de alto contraste.

### 2. Autenticación y Multi-Tenant (Supabase Auth)
- **Inicio de Sesión y Registro:** Pantalla de autenticación moderna con efecto de glassmorphic dark mode.
- **Aprovisionamiento Automático:** Se implementó un trigger PostgreSQL en base de datos (`on_auth_user_created`) que, al registrarse un usuario:
  - Crea un perfil asociado en la tabla `profiles`.
  - Crea una compañía por defecto (Tenant) asignada al perfil.
  - Otorga de forma automática el rol de **Administrador**.

### 3. CRM de Clientes (Básico)
- **Tabla de Datos (PC):** Data grid avanzado con ordenamiento y filtros por estado y categoría del cliente.
- **Tarjetas de Clientes (Móvil):** Formato vertical simplificado con botones directos de acción (Llamar, ver ubicación).
- **Seguridad RLS (Row Level Security):** Filtro automático a nivel de base de datos para que los empleados de una compañía solo vean los clientes creados dentro de su propia organización.
